Get Running in 30 Seconds

1. Clone & Install
git clone https://github.com/juggperc/uupinfetch.git
cd uupinfetch
pip install -r requirements.txt
2. Start Server
./start.sh    # Linux/Mac
start.bat    # Windows
# or: uvicorn app.main:app --reload
3. Query from Your Bot
curl http://localhost:8000/api/v1/items/search?q=AK-47
